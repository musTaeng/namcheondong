# 사장남천동 커뮤니티 웹사이트

## 개요

이 프로젝트는 유튜브 채널 "사장남천동"의 팬들을 위한 커뮤니티 웹사이트 "익하이브"입니다. 사용자들은 이 웹사이트를 통해 소통하고 정보를 공유할 수 있습니다.

## 스타일, 디자인 및 기능

### 초기 버전

*   **디자인:**
    *   모던하고 깔끔한 디자인.
    *   웹사이트 상단에 탐색 메뉴가 포함된 헤더.
    *   반응형 레이아웃으로 모바일과 데스크톱 환경을 모두 지원.
*   **기능:**
    *   자유게시판, 익현이지도 등 다양한 게시판 메뉴 구성.

### 테마 모드 지원 추가 (2026-03-10)

*   **다크/라이트 모드:**
    *   사용자가 테마를 토글할 수 있는 버튼 추가.
    *   `localStorage`를 사용하여 사용자의 테마 선호도를 저장.
    *   시스템의 다크 모드 설정을 감지하여 초기 테마 설정.

### 제휴 문의 폼 추가 (2026-03-10)

*   **Formspree 연동:**
    *   백엔드 코드 없이 이메일로 문의를 받을 수 있는 Formspree 서비스 연동.

### 게시판 라우팅 및 Disqus 댓글 연동 (2026-03-10)

*   **해시 기반 라우팅:**
    *   `main.js`에서 `window.location.hash`를 감지하여 페이지 전환 없이 게시판 내용을 업데이트함.
    *   각 메뉴 아이콘 클릭 시 해당 게시판으로 즉시 전환.
*   **Disqus 댓글 기능:**
    *   각 게시판 하단에 Disqus 댓글 창을 삽입하여 사용자 간 소통 기능 구현.
    *   게시판 전환 시 `DISQUS.reset()`을 호출하여 해당 게시판 전용 댓글 스레드를 로드함.
*   **UI 개선:**
    *   Sticky Header 적용으로 네비게이션 편의성 증대.
    *   게시판별 헤더 디자인 적용.

## 현재 계획

1.  각 게시판의 특성에 맞는 기본 안내 문구 및 템플릿 추가.
2.  사용자 참여를 유도하기 위한 실시간 공지사항 배너 추가.
